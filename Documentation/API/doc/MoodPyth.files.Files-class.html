get_files(self, contextid, component, filearea, itemid, filepath, filename, modified='')

source code 

Browse moodle files. Do not download any content.

Parameters:
  • contextid (Integer) - context identifier.
  • component (String) - component.
  • filearea (String) - file area.
  • itemid (Integer) - item identifier.
  • filepath (String) - file path.
  • filename (String) - file name.
  • modified (Integer) - timestamp to return files changed after this time (Default to "").

upload_file(self, path, moodlepath)

source code 

Uploads a file to the users private files area.

Parameters:
  • path (Integer) - file path to be uploaded.
  • moodlepath (Integer) - moodle private files area path. It must start and end with the "/" character.
Raises:
  • IOError - if file specified in path parameter do not exists or cannot be readable.
