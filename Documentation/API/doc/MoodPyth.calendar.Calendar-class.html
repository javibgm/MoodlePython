create_calendar_events(self, events)

source code 

Create calendar events

Parameters:
  • events (List of Dictionary) - 1 or more events to create:
    • name (string) - event name
    • description (string) Default to "null" - event description
    • format (int) Default to "1" - description format (1 = HTML, 0 = MOODLE, 2 = PLAIN or 4 = MARKDOWN)
    • courseid (int) Default to "0" - course id
    • groupid (int) Default to "0" - group id
    • repeats (int) Default to "0" - number of repeats
    • eventtype (string) Default to "user" - Event type
    • timestart (int) Default to actual time - starting event time
    • timeduration (int) Default to "0" - event duration
    • visible (int) Default to "1" - visibility
    • sequence (int) Default to "1" - sequence
Returns:
A dictionary with the events created and possible warnings:
  • events (List of Dictionary) - list of events created:
    • id (int) - event id
    • name (string) - event name
    • description (string) Optional - event description
    • format (int) - description format (1 = HTML, 0 = MOODLE, 2 = PLAIN or 4 = MARKDOWN)
    • courseid (int) - course id
    • groupid (int) - group id
    • userid (int) - user id
    • repeatid (int) Optional - repeat id
    • modulename (string) Optional - module name
    • instance (int) - instance id
    • eventtype (string) - Event type
    • timestart (int) - timestart
    • timeduration (int) - time duration
    • visible (int) - visible
    • uuid (string) Optional - unique id of ical events
    • sequence (int) - sequence
    • timemodified (int) - time modified
    • subscriptionid (int) Optional - Subscription id
  • warnings (List of Dictionary) Optional - list of possible warnings for each event created:
    • item (string) Optional - item
    • itemid (int) Optional - item id
    • warningcode (string) - the warning code can be used by the client app to implement specific behavior
    • message (string) - untranslated English message to explain the warning
Raises:
  • TypeError - if events input parameter type is not a list or is an empty list.

get_calendar_events(self, eventids=[], courseids=[], groupids=[], userevents=1, siteevents=1, timestart=0, timeend='', ignorehidden=1)

source code 

Get calendar events.

Parameters:
  • eventids (List of Integer) - 0 or more event ids
  • courseids (List of Integer) - 0 or more course ids for which events will be returned
  • groupids (List of Integer) - 0 or more group ids for which events should be returned
  • userevents (Integer) - Set to true to return current user's user events
  • siteevents (Integer) - Set to true to return global events
  • timestart (Integer) - Time from which events should be returned
  • timeend (Integer) - Time to which the events should be returned
  • ignorehidden (Integer) - Ignore hidden events or not
Returns:
a dictionary with the events which match with the search parameters and warnings.
Raises:
  • TypeError - if eventids, courseids or groupids input parameters types are not lists.

Note: This function only returns events that have been created with MoodPyth
